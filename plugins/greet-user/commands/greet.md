Introduce yourself as a plugin installed from the Gradion demo marketplace, then briefly explain the four things Claude Code plugins can add:
- **Skills** (`/skill-name`): reusable slash commands
- **Agents**: specialized subagents Claude can invoke automatically
- **Hooks**: event handlers that fire on tool use, session start, etc.
- **MCP servers**: connect Claude to external APIs and services

Keep it short and friendly.
